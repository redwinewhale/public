### 데이터 입력 및 '기본 수식' ( = 와 셀 참조)
-   특정 셀 참조 방법
    -   `=B2+C2` : B2의 값과 C2의 값을 더한 값을 넣겠다

### 채우기 핸들

-   우측 하단 점을 끌어내리면 다른 칸은 자동 참조
    >   ex: `D3` `=B3+C3`

### 기본 함수`(SUM, AVERAGE, MAX, MIN, COUNT)`

-   SUM
    -   지정 범위의 합계
    >   ex: `=SUM(B2:B4)`

-   AVERAGE
    -   지정 범위의 평균
    >   ex: `=AVERAGE(B2:B4)`

-   MAX
    -   지정 범위의 최댓값
    >   ex: `=MAX(B2:B4)`

-   MIN
    -   지정 범위의 최솟값
    >   ex: `=MIN(B2:B)`

-   COUNT
    -   지정 범위의 개수(숫자가 있는 셀의 개수)
    >   ex: `=COUNT(B2:B4)`

---
채우기 핸들로 C 라인도 한번에 작성 가능

### 절대 참조($)

-   정대 참조 사용시 채우기 핸들로 복사해도 셀 주소가 변하지 않음
    -   고정하고자 하는 셀 주소에 `$` 사용
        >   ex: `=B2+$F$2`  
        >   셀 주소 입력후 `F4` 누르면 자동으로 `$` 입력
