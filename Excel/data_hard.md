### 조건부 서식

-   데이터에 조건을 걸어 조건에 맞는 셀에만 자동으로 서식(색상, 막대)을 입히는 기능

1.  범위 선택 후 서식의 조건부 서식 선택
2.  조건과 서식 지정

### IF 함수

-   조건 명령
    -   `=IF(B2="영업팀", "지급", "미지급")`
        >   B2="영업팀" : 조건  
        >   "지급" : 조건이 참일 경우 표시할 값
        >   "미지급" : 조건이 거짓일 경우 표시할 값
    -   `=IF(B2="영업팀", E2+500, "미지급")`
        >   다음과 같이 수식도 삽입 가능

### VLOOKUP 함수 (찾기/참조 함수)

-   별도의 표에서 특정 값을 찾아와 현재 표에 채워 넣는 기능
    -   `=VLOOKUP(C2, $H$2:$I$4, 2, FALSE)`
        >   `C2`: (찾을 값) G2 셀의 직급을 C2 기준으로 찾음  
        >   `$H$2:$I$4`: (참조할 표 범위) H2부터 I4까지의 표에서 찾음  
        >   `2`: (가져올 열 번호) 참조 표(H:I)에서 2번째 열에 있는 성과급 값을 가져옴  
        >   `FALSE`: (일치 옵션) 반드시 직급이 정확하게 일치하는 값만 찾음(거의 항상 FALSE)

-   TRUE 옵션
    -   근사값 찾기(작거나 같은 값중 가장 큰값을 찾음)
        >   예: 점수 등급  
        >   학생의 점수가 85점이고 등급표를 참조할 경우 등급표에 80점이 B 90점이 A라 나와있으면 80점인 B를 가져옴


### 피벗 테이블

-   마우스로 쓸 수 있는 데이터 요약/분석 도구

1.  표 선택후 피벗 테이블 선택
2.  원하는 값들을 넣어 데이터 요약 및 분석 가능
```
#   부서별 평균 연봉

1.  부서를 마우스로 끌어 아래 행에 넣기

2.  연봉(만원)을 끌어 값 상자에 넣기

3.  요약기준을 SUM(합계) 대신 AVERAGE(평균)을 선택

->  부서별 평균 연봉 확인 가능

#   부서별, 직급별 인원수

1.  부서를 열 상자로 이동

2.  직급을 행 상자에 넣기

3.  이름을 값 상자에 넣기

->  부서, 직급별 인원수 확인 가능
```
